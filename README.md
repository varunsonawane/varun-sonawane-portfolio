# Varun Sonawane — Personal Portfolio
 
A personal portfolio website showcasing my experience, projects, and skills in data engineering, cloud architecture, and AI systems.
 
**Live site:** [vson.vercel.app](https://vson.vercel.app)
 
---
 
## About
 
Built with Next.js and TypeScript, this portfolio highlights my journey from a B.E. in Information Technology (Savitribai Phule Pune University) to an M.S. in Computer Science at Indiana University Bloomington (GPA 3.83/4.0). It covers my work across data pipelines, cloud infrastructure, AI/ML systems, and hackathon projects.

## Tech Stack
 
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS, CSS Modules

## Getting Started
 
### Prerequisites
 
- Node.js 18+
- pnpm (recommended) or npm
### Installation
 
```bash
git clone https://github.com/varunsonawane/varun-sonawane-portfolio.git
cd varun-sonawane-portfolio
pnpm install
```
 
### Development
 
```bash
pnpm dev
```
 
Open [http://localhost:3000](http://localhost:3000) in your browser.
 
### Build
 
```bash
pnpm build
pnpm start
```
